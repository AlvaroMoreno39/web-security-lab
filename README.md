# Web Security Lab

Learning project to practice and understand common web vulnerabilities
in a controlled local environment.

## Environment
- Ubuntu Server VM
- Apache + PHP + MySQL
- Access via SSH and VS Code

## Vulnerabilities Covered
- Reflected XSS
- Stored XSS
- DOM-based XSS
- SQL Injection (login & inserts)
- Basic API usage with prepared statements

## Goal
The purpose of this repository is to document my hands-on learning
process in web security, focusing on understanding:
- why vulnerabilities occur
- their impact
- and how to mitigate them properly

This is an educational project for learning purposes.
